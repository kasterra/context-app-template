# 이건 뭔가요?

React Context API를 알아보자는 주제로 글을 쓴 이 레포 주인장의 블로그에서 사용하는 교보재 파일들이 담긴 레포입니다.

`CRA --template typescript`로 후딱 만들고, Context 부분을 제거해서 템플릿만 남겨두었습니다. 이제 이 파일을 다운받아서 Context를 예쁘게 적용시켜 봅시다.
